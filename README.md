# TrainTicket

Mobile app (IOS/Android) for Train Ticket Management System.

> [!NOTE]
> This project is for educational purposes only, not for commercial purposes

## Tech stack
- React Native
- React Navigation
- React Query
- React Hook Form
- Zustand

## Requirements
- Node.js 18 >=
- Android Studio (for Android development)
- Xcode (for iOS development)
- TrainTicket backend 🐧
